# 🏠 F4Life - Website Institucional

Projeto de um site institucional desenvolvido com HTML, CSS e JavaScript, simulando uma plataforma de aluguel de moradias para universitários.

---

## 📄 Páginas do projeto
- Página inicial
- Quem somos
- Fale conosco (com formulário)

---

## 🚀 Funcionalidades
- Navegação entre páginas
- Formulário de contato com validação
- Layout responsivo
- Organização de conteúdo com HTML e CSS

---

## 💻 Tecnologias utilizadas
- HTML
- CSS
- JavaScript

---

## 🎯 Objetivo
Este projeto foi desenvolvido com o objetivo de praticar:
- Criação de sites multi-páginas
- Estruturação de conteúdo
- Estilização com CSS
- Manipulação básica com JavaScript
