# F4Life - Website Institucional

Projeto de um site institucional desenvolvido com HTML e CSS, simulando uma plataforma de aluguel de moradias para universitários.

## Páginas do projeto

- Página inicial
- Quem somos
- Fale conosco (com formulário)

## Funcionalidades

- Navegação entre páginas
- Formulário de contato
- Layout organizado com HTML e CSS

## Tecnologias utilizadas

- HTML
- CSS
- JavaScript 

## Objetivo

Este projeto foi desenvolvido para praticar a criação de sites multi-páginas, estruturação de conteúdo e organização de layout.
